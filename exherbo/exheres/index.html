Cave fetches,unpacks and prepares the compilation environment from package templates, exheres-0. Because of this, this allows for options enabling for packages. For example, firefox has an option for pulseaudio. If pulseaudio integration is desired, then the user will enable the option within /etc/paludis/options.conf by including the line net-www/firefox pulseaudio.

In order to fetch the package, in a tarball format, a download link is required. For example, for the emacs tarball, the DOWNLOADS variable is set equal to https://mirror.us-midwest-1.nexcess.net/gnu/emacs/emacs-27.2.tar.gz, thus DOWNLOADS="https://mirror.us-midwest-1.nexcess.net/gnu/emacs/emacs-27.2.tar.gz".
