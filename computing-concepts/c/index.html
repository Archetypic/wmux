Modern C

Takeways (literal)

  • Takeaway A C and C++ are different: don't mix them, and don't mix them up.
  • Takeaway B Don't panic.
  • Takeaway 0.1.1.1 C is an imperative programming language.
  • Takeaway 0.1.2.1 C is a compiled program.
  • Takeaway 0.1.2.2 A correct C program is portable between different platforms.
  • Takeaway 0.1.2.3 A C program should compile cleanly without warnings.
  • Takeaway 0.2.1.1 Punctuation characters can be used with several different meanings.
  • Takeaway 0.2.2.1 All identifiers in a program have to be declared.
  • Takeaway 0.2.2.2 Identifiers may have several consistent declarations.
  • Takeaway 0.2.2.3 Declarations are bound to the scope in which they appear.
  • Takeaway 0.2.3.1 Declarations specify identifiers, whereas definitions specify objects.
  • Takeaway 0.2.3.2 An object is defined at the same time it is initialized.
  • Takeaway 0.2.3.3 Missing elements in intializers default to 0.
  • Takeaway 0.2.3.4 For an array with n elements, the first element has index 0, and the last has index n-1.
  • Takeaway 0.2.3.5 Each object or function must have exactly one definition.
  • Takeaway 0.2.4.1 Domain iterations should be coded with a for statement.
  • Takeaway 0.2.4.2 The loop variable should be be defined in the initial part of a for.

Since C is compiled programming language, a compilier is required to translate theC text into binary code or an execuatable. Examples of compilers are gcc, clang, and c99.

Punctuation in C

  • Bracket: { ... }
  • Bracket: ( ... )
  • Bracket: [ ... ]
  • Bracket: /* ... */
  • Bracket: < ... >
  • Terminator: ,
  • Terminator: ;
Referrering C program

Brackets are always paired. /* ... */ is the comment sytax. Comments are ignored by the compiler. The fixed values in a program are known as literals. Indentifiers are names given to certain entities. i is an example of a data object or variable. Type specify "sort" of a new object, typically appended with _t. main and printf are functions. Constants, such as EXIT_SUCCESS. main must always be present, because it is the staring point of any program's execution. Operators in C can be utilized for initialization and assignment, comparison, increment and decrment a variable, and perform arithmetic on numbers.

A declaration specifies what the identifier represents. An array is an aggregate of several items of the same type. Variables are named items that store values. The scope is a part of the program where an identifier is visible. A block scope is delimited by a block such as { ... }. A file scope is not delimited by { ... } and its identifiers are referred to as globals. An intialization is a grammatical construct that augments a declaration and provides an initial value for the object. Statements are instructions that tell the compilier what to do with idenifiers that have been declared thus far.

Within a for-statement, the code that is repeated is called the loop body; it is the { ... } block that follows the for ( ... ). Inside of the ( ... ) contains the declaration, definition, and initializer. The loop condition, i < 5, specifies the duration the for iteration should continue. It thus keeps iterating as long as i is less than 5. ++i is executed after each iteration. It increases the value of i by 1 each iteration.

Function calls are special statements that suspend the execution of the current function and then hand over control to the named function. Function return tells main function to return to the statement that it was called from once it's done.
