The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
<h1–h6> headings include a heading (levels 1–6)
<p> paragraph include a paragraph of text
<table> table include a table
<tr> table header make a table header
<td> table data include a table data cell
<body> Body Everything inside this element is shown inside the main browser window.
<head> Head Contains information about the page as opposed to shown within the main browser window.
<title> Title Contents of <title> are shown on tab for that page.
<blockquote> Blockquote Used for longer quotes that take up an entire paragraph.
<address> Address Contains the contact details for the author of the page.
<ol> Ordered list Creates an ordered list.
<li> List item Creates subcomponent for lists.
<ul> Unordered list Creates and unordered list.
<dl> Definition list Creates a definition list.
<dt> Defining term Contains the defining term.
<dd> Definition Contains the definition.
<figure> Figure Contains images and their associated captions.
<figcaption> Caption for figures Adds captions to image in <figure>.
<form> Form Allows user to interact with webpage.
<fieldset> Fieldset Allows for grouping of form related controls congruently.
<legend> Legend Contains a caption that identifies the purpose of that group of form control.
<!-- --> Comment Contains a comment.
<div> Divider Group a set of elements together in one block-level box.
<iframe> Inline frame Embed an html page from another website on the current page.
<meta> Meta data Contains information about that webpage.
<video> Video Shows a video on page.
<source> Source Specifies source of audio/video.
<audio> Audio Play audio file on page.

Inline Elements

Tag Name Purpose Example Result
<em> empahsized make emphasized text <em>technical sophistication</em> technical sophistication
<strong> strong make strong text <strong>at least a billion people</strong> at least a billion people
<a> anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
<img> image include an image <img src="http://bit.ly/1MZAFuQ" alt="Micheal Hartl">
<b> Bold Bold the interior text. <b>BOLD</b> This is bold.
<i> Italics Italicize the interior text. <i>ITALICS</i> This is italicized text.
<sup> Superscript Makes string a superscript. <sup>High Level</sup> 22=4
<sub> Subscript Make string a subscript. <sub>Low Level.</sub> CO2
<br> Break line Adds a line break. <br> This is the first line.
This is the second.
<hr> Horizontal rule Add a horizontal rule. <hr> This is the first line.
This is the second line.
<q> Quote Quotes interior text. <q>The Quote</q> I said this.
<abbr> Abbreviation Specifies an abbreviation. <abbr title="National Aeronatics and Space Administration">NASA</abbr> Prof
<cite> Citation Cite a piece of literature. <cite>War and Peace</cite> War and Peace
<dfn> Defining instance Indicates the defining instance of a new term. <dfn>Unknown word</dfn> The Unknown word is confusing.
<ins> Insert Shows text that has been inserted into the document. The <ins>best</ins> idea. The best idea.
<del> Delete Show text that has been deleted from the document. The <del>worst</del> idea. The worst idea.
<s> Strikethrough Indicates that corresponding text is no longer accurate or relevant but should not be deleted. The price is now <s>$999</s> $899. The price is now $999 $899.
<textarea> Text area Creates multi-line input text. <textarea name="comments" cols="20" rows="4">Enter your comments...</textarea>
<select> Select Select one option from a drop down list. <select name="device"> <option value="ipod">iPod</option></select>
<button> Button Allows user more control of button appearance and allow interior elements.
<label> Label Labels form controls. <label>Age: <input type="text" name="age" />lt/label>
<span> Span An inline equivalent of <div>.

Table Elements

Tag Name Purpose
<table> Table Creates a table.
<tr> Table row Creates a table row.
<td> Table data Insert data into a table cell.
<th> Table header Insert data for column header or row header.

Long table

Tag Name Purpose
<thead> Table header Insert heading of long table here.
<tbody> Table body Insert body of long table here.
<tfoot Table footer Insert footer of long table here.

Input attributes

Attribute Purpose
type="text" Creates a single-line text input.
type="password" Similar to type="text" but input characters are blocked out.
type="radio" Allows user to select on of multiple options.
type="checkbox" Allows user to select and deselect multiple options.
type="file" Allows user to browse file system and select a file.
type="submit" Sends a form to the server.
type="image" Use an image for the submit button.
type="hidden" Shows a hidden form control.
type="date" Sets input to a calender value.
type="email" Sets input to email address.
type="url" Sets input to URL.
type="search" Allows for search queries.
