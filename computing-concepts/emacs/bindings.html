Emacs Bindings

Command Default Key Binding Description Command format
forward-char C-f Move point N characters forward (forward-char &optional N)
backward-char C-b Move point N characters backward (backward-char &optional ARG)
forward-word M-f
ESC <right>		 Move point ARG words forward (forward-ward &optional ARG)
backward-word M-b
ESC <left>		 Move point ARG words backward (backward-word &optional ARG)
next-line C-n
<down>		 Move cursor vertically down ARG lines (next-line &optional ARG TRY-VSCROLL)
previous-line C-p
<up>		 Move cursor vertically up ARG lines. (previous-line &optional ARG TRY-VSCROLL)
move-beginning-of-line C-a
<home>		 Move point to beginning of current line as displayed. (move-beginning-of-line ARG)
move-end-of-line C-e
<end>		 Move point to end of current line as displayed. (move-end-of-line ARG)
backward-sentence M-a Move backward to start of sentence. (backward-sentence &optional ARG)
forward-sentence M-e Move forward to next end of sentence. (forward-sentence &optional ARG)
beginning-of-buffer <begin>
<C-home<
M->		 Move point to the beginning of the buffer. (beginning-of-buffer &optional ARG)
end-of-buffer <C-end>
M->		 Move point to end of the buffer. (end-of-buffer &optional ARG)
universal-argument C-u Begin a numeric argument for the following command. (universal-argument)
keyboard-quit C-g Signal a 'quit' condition. (keyboard-quit)
delete-other-windows C-x 1 Make WINDOW fill its frame. (delete-other-windows &optional WINDOW)
backward-kill-word <C-backspace>
M-<DEL>		 Kill characters backward unitl encountering the beginning of a word. (backward-kill-word ARG)
kill-word <C-delete>
M-d		 Kill characters forward until encountering the end of a word. (kill-word ARG)
kill-line C-k
<deleteline>		 Kill the rest of the current line; if no nonblanks there, kill thru newline. (kill-line &optional ARG)
kill-sentence M-k Kill from point to end of sentence. (kill-sentence &optional ARG)
set-marker-command C-@
C-SPC		 Set the mark where point is and activate it; or jump to the mark. (set-mark-command ARG)
kill-region C-w
<S-delete>		 Kill ("cut") text between point and mark. (kill-region BEG END &optional REGION)
yank C-y
<S-insertchar>		 Reinsert ("paste") the last strech of killed text. (yank &optional ARG)
undo C-_
<undo>
C-/
C-x u		 Undo some previous changes (undo &optional ARG)
find-file <open>
C-x C-f		 Edit file FILENAME. (find-file FILENAME &optional WILDCARDS)
save-buffer C-x C-s Save current buffer in visited file if modified. (save-buffer &optional ARG)
list-buffers C-x C-b Display a list of existing buffers. (list-buffers &optional ARG)
switch-to-buffer C-x b Display buffer BUFFER-OR-NAME in the selected window. (switch-to-buffer BUFFER-OR-NAME &optional NORECORD FORCE-SAME-WINDOW)
save-buffers-kill-terminal C-x C-c Offer to save each buffer, then kill the current connection. (save-buffers-kill-terminal &optional ARG)
