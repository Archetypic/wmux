Most essential commands for mastering GNU Emacs

Command Default Key Binding Description Command format
describe-mode C-h m Display documentation of current major mode and minor modes. (describe-mode &optional BUFFER)
apropos-command C-h a Show commands (interactively callable functions that match PATTERN. (apropos-command PATTERN &optional DO-ALL VAR-PREDICATE)
describe-key-briefly C-h c Print the name of the functions KEY-LIST invokes. (describe-key-briefly KEY-LIST &optional INSERT)
describe-key C-h k Display documentation of the function invoked by KEY-LIST. (describe-key KEY-LIST &optional BUFFER)
describe-function C-h f Display the full documentation of FUNCTION (a symbol). (describe-function FUNCTION)
info C-h i Enter Info, the documentation browser. (info &optional FILE-OR-NODE BUFFER)
info-emacs-manual C-h r Display the Emacs manual in Info mode. (info-emacs-manual)
help-with-tutorial C-h t Select the Emacs learn-by-doing tutorial (help-with-tutorial &optional ARG DONT-ASK-FOR-REVERT)
