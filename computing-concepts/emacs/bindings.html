Command Default Key Binding Description Command format
forward-char C-f Move point N characters forward (forward-char &optional N)
backward-char C-b Move point N characters backward (backward-char &optional ARG)
forward-word M-f
ESC <right>		 Move point ARG words forward (forward-ward &optional ARG)
backward-word M-b
ESC <left>		 Move point ARG words backward (backward-word &optional ARG)
next-line C-n
<down>		 Move cursor vertically down ARG lines (next-line &optional ARG TRY-VSCROLL)
previous-line C-p
<up>		 Move cursor vertically up ARG lines. (previous-line &optional ARG TRY-VSCROLL)
move-beginning-of-line C-a
<home>		 Move point to beginning of current line as displayed. (move-beginning-of-line ARG)
move-end-of-line C-e
<end>		 Move point to end of current line as displayed. (move-end-of-line ARG)
backward-sentence M-a Move backward to start of sentence. (backward-sentence &optional ARG)
forward-sentence M-e Move forward to next end of sentence. (forward-sentence &optional ARG)
back-to-indentation M-m Move point to the first non-whitespace character on this line. (back-to-indentation)
beginning-of-buffer <C-home>
M->		 Move point to the beginning of the buffer. (beginning-of-buffer &optional ARG)
end-of-buffer <C-end>
M->		 Move point to end of the buffer. (end-of-buffer &optional ARG)
forward-sexp C-M-f Move forward across one balanced expression (sexp). (forward-sexp &optional ARG)
backward-sexp C-M-b Move backward across one balanced expression (sexp). (backward-sexp &optional ARG)
down-list C-M-d Move forward down one level of perentheses. (down-list &optional ARG)
backward-up-list C-M-u Move backward out of one level of parentheses. (backward-up-list &optional ARG ESCAPE-STRINGS NO-SYNTAX-CROSSING)
forward-list C-M-n Move forward across one balanced group of parentheses. (forward-list &optional ARG)
backward-list C-M-p Move backward across one balanced group of parentheses. (backward-list &optional ARG)
forward-paragraph M-} Move forward to end of paragraph. (forward-paragraph &optional ARG)
backward-paragraph M-{ Move backward to start of paragraph. (backward-paragraph &optional ARG)
end-of-defunc C-M-e Move forward to next end of defun. (end-of-defun &optional ARG)
beginning-of-defun C-M-a Move backward to the beginning of a defun. (beginning-of-defun &optional ARG)
forward-page C-x ] Move forward to page boundary. (forward-page &optional COUNT)
backward-page C-x [ Move backward to page boundary. (backward-page &optional COUNT)
scroll-up-command C-v Scroll text of selected window upward ARG lines; or near full screen if no ARG. (scroll-up-command &optional ARG)
scroll-down-command M-v Scroll text of selected window down ARG lines; or near full screen if no ARG. (scroll-down-command &optional ARG)
scroll-other-window C-M-v Scroll next window upward ARG lines; or near full screen if no ARG. (scroll-other-window &optional ARG)
scroll-other-window-down C-M-S-v Scroll next window downward ARG lines; or near full screen if no ARG. (scroll-other-window-down &optional ARG)
beginning-of-buffer M-< Move point to the beginning of the buffer. (beginning-of-buffer &optional ARG)
end-of-buffer M-> Move point of the end of the buffer. (end-of-buffer &optional ARG)
universal-argument C-u Begin a numeric argument for the following command. (universal-argument)
keyboard-quit C-g Signal a 'quit' condition. (keyboard-quit)
delete-other-windows C-x 1 Make WINDOW fill its frame. (delete-other-windows &optional WINDOW)
backward-kill-word <C-backspace>
M-<DEL>		 Kill characters backward unitl encountering the beginning of a word. (backward-kill-word ARG)
kill-word <C-delete>
M-d		 Kill characters forward until encountering the end of a word. (kill-word ARG)
kill-line C-k Kill the rest of the current line; if no nonblanks there, kill thru newline. (kill-line &optional ARG)
kill-sentence M-k Kill from point to end of sentence. (kill-sentence &optional ARG)
set-marker-command C-@
C-SPC		 Set the mark where point is and activate it; or jump to the mark. (set-mark-command ARG)
exchange-point-and-mark C-x C-x Put the mark where point is now, and point where the mark is now. (exchange-point-and-mark &optional ARG)
mark-paragraph M-h Put point at beginning of this paragraph, mark at end. (mark-paragraph &optional ARG ALLOW-EXTEND)
mark-whole-buffer C-x h Put point at beginning and mark at end of buffer. (mark-whole-buffer)
mark-defun C-M-h Put mark at end of this defun, point at beginning. (mark-defun &optional ARG)
mark-page C-x C-p Put mark at end of page, point at beginning. (mark-page &optional ARG)
mark-word M-@ Set mark ARG words away from point. (mark-word &optional ARG ALLOW-EXTEND)
mark-sexp C-M-@
C-M-SPC		 Set mark ARG sexps from point. (mark-sexp &optional ARG ALLOW-EXTEND)
kill-region C-w
<S-delete>		 Kill ("cut") text between point and mark. (kill-region BEG END &optional REGION)
yank C-y Reinsert ("paste") the last strech of killed text. (yank &optional ARG)
undo C-_
C-/
C-x u		 Undo some previous changes (undo &optional ARG)
find-file C-x C-f Edit file FILENAME. (find-file FILENAME &optional WILDCARDS)
save-buffer C-x C-s Save current buffer in visited file if modified. (save-buffer &optional ARG)
list-buffers C-x C-b Display a list of existing buffers. (list-buffers &optional ARG)
switch-to-buffer C-x b Display buffer BUFFER-OR-NAME in the selected window. (switch-to-buffer BUFFER-OR-NAME &optional NORECORD FORCE-SAME-WINDOW)
save-buffers-kill-terminal C-x C-c Offer to save each buffer, then kill the current connection. (save-buffers-kill-terminal &optional ARG)
kill-buffer C-x k Kill the buffer specified by BUFFER-OR-NAME. (kill-buffer &optional BUFFER-OR-NAME)
keyboard-escape-quit ESC ESC ESC Exit the current "mode" (in a generalized sense of the word). (keybaord-escape-quit)

delete-window C-x 0 Delete WINDOW. (delete-window &optional WINDOW)
split-window-below C-x 2 Split the selected window into two windows, one about the other. (split-window-below &optional SIZE)
split-window-right C-x 3 Split the selected window into two side-by-side windows. (split-window-right &optional SIZE)
other-window C-x o Select another window in cyclic ordering of windows. (other-window COUNT &optional ALL-FRAMES)
find-file-other-window C-x 4 C-f
C-x 4 f		 Edit file FILENAME, in another window (find-file-other-window FILENAME &optional WILDCARDS)
dired-other-window C-x 4 d "Edit" directory DIRNAME. Like 'dired; but selects in another window. (dired-other-window DIRNAME &optional SWITCHES)
display-buffer C-x 4 C-o Display BUFFER-OR-NAME in some window, without selecting it. (display-buffer BUFFER-OR-NAME &optional ACTION FRAME)
display-buffer-other-frame C-x 5 C-o Display buffer BUFFER preferably in another frame. (display-buffer-other-frame BUFFER)
switch-to-buffer-other-window C-x 4 b Select the buffer specified by BUFFER-OR-NAME in another window. (switch-to-buffer-other-window BUFFER-OR-NAME &optional NORECORD)
kill-buffer-and-window C-x 4 0 Kill the current buffer and delete the selected window. (kill-buffer-and-window)
make-frame-command C-x 5 2 Make a new frame, on the same terminal as the selected frame. (make-frame-command)
switch-to-buffer-other-frame C-x 5 b Switch to buffer BUFFER-OR-NAME in another frame. (switch-to-buffer-other-frame BUFFER-OR-NAME &optional NORECORD)
delete-frame C-x 5 0 Delete FRAME, permanently eliminating it from use. (delete-frame &optional FRAME FORCE)
delete-other-frames C-x 5 1 Delete all frames on FRAME's terminal, except FRAME. (delete-other-frames &optional FRAME)
find-file-other-frame C-x 5 C-f Edit file FILENAME, in another frame. (find-file-other-frame FILENAME &optional WILDCARDS.)
dired-other-frame C-x 5 d "Edit" directory DIRNAME. Like 'dired' but makes a new frame. (dired-other-frame DIRNAME &optional SWITCHES)
bookmark-set C-x r m Set a bookmark named NAME at the current location. (bookmark-set &optional NAME NO-OVERWRITE)
bookmark-bmenu-list C-x r l Display a list of existing bookmarks. (bookmark-bmenu-list)
bookmark-jump C-x r b Jump to a bookmark BOOKMARK (a point in some file). (bookmark-jump BOOKMARK &optional DISPLAY-FUNC)
number-to-register C-x r n Store a number in a register. (number-to-register NUMBER REGISTER)

C-x r x		 Copy region into register REGISTER. (copy-to-register REGISTER START END &optional DELETE-FLAG REGION)

jump-to-register C-x r j Move point to location stored in a register. (jump-to-register REGISTER &optional DELETE)

Most essential commands for mastering GNU Emacs

Command Default Key Binding Description Command format
describe-mode C-h m Display documentation of current major mode and minor modes. (describe-mode &optional BUFFER)
apropos-command C-h a Show commands (interactively callable functions that match PATTERN. (apropos-command PATTERN &optional DO-ALL VAR-PREDICATE)
describe-key-briefly C-h c Print the name of the functions KEY-LIST invokes. (describe-key-briefly KEY-LIST &optional INSERT)
describe-key C-h k Display documentation of the function invoked by KEY-LIST. (describe-key KEY-LIST &optional BUFFER)
describe-function C-h f Display the full documentation of FUNCTION (a symbol). (describe-function FUNCTION)
info C-h i Enter Info, the documentation browser. (info &optional FILE-OR-NODE BUFFER)
info-emacs-manual C-h r Display the Emacs manual in Info mode. (info-emacs-manual)
help-with-tutorial C-h t Select the Emacs learn-by-doing tutorial (help-with-tutorial &optional ARG DONT-ASK-FOR-REVERT)
